# Aurora 轻量级 Go Web 框架

Aurora 是一个用 Go (Golang) 编写的 Web 框架 , 开箱即用。如果您需要性能和良好的生产力，您一定会喜欢 Aurora 。



## 主要内容

- [x] 基础路由
- [x] 路径参数
- [x] 文件上传
- [x] 分组路由
- [x] 拦截器
- [x] 静态文件
- [x] 服务器HTML渲染
- [x] 重定向
- [x] 服务转发
- [x] 邮件支持
- [x] GORM 集成
- [x] Redis 集成
- [x] gRPC 端口整合
